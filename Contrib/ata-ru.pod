=head1 ИМЯ

ata - Apache Transfer-log Analyzer.

=head1 ОПИСАНИЕ

ata - скрипт для генерации отчета о количестве запросов и размере трафика.

=head1 СИНТАКСИС

ata <ОПЦИИ>

=head1 ОПЦИИ

=over 10

=item -file < file_name >

имя log файла.

=item -format < regexp >

регулярное выражение для разбора лог файла.

=item -f < field_name >

имена полей в лог файле. Количество `()` в регулярном выражении должно соответствовать количеству параметров `-f`. Параметр можно использовать многократно.

=item -gb < field_name >

group by... Имена полей по которым производится группировка в отчете. Любые имена из `-f`. Для примера `... -gb req_user -gb method` группировать по req_user ( /~user_name ) и method ( GET || POST | HEAD ... etc ).  Параметр можно использовать многократно. Для большей информации смотрите примеры.

=item -order_by < field_name >

order by... Имена полей для сортировки отчета. Любые имена из `-gb` и два зарезервированных слова: 'count' - количество запросов, 'size' - размер трафика. Параметр можно использовать многократно. 

=item -txt

отдать отчет в текстовом формате.

=item -help

помощь.

=item -unit < unit_name >

единица измерения трафика ( B, K, M, G - Байты, Килобайты, Мегабайты, Гигабайты ).

=item -tr_size_fld < field_name >

имя поля ( из `-f` ) для определения количества трафика.

=item -show

показать процесс отработки лог файла.

=item -dump < log_file_name >

сохранить нераспознанные строки из лог файла апача в данном файле.

=back

=head1 Примечания

'Count' и 'Size' - зарезервированные слова, не используйте их в `-f`.

Расчет процентов производится только при генерации отчета в html формате.

Регулярное выражение ( `-format` ) обрабатывает строки следующего
формата ( по умолчанию ) :

 66.28.210.71 gosha - [15/Jun/2002:10:19:21 +0400] "GET /~alex/cgi-bin/view.cgi?id=12 HTTP/1.0" 200 3087

`-f` содержит следующие поля ( для ^^^ данной строки, по умолчанию ):

 host		= 66.28.210.71
 login		= gosha
 user		= -
 datetime	= 15/Jun/2002:10:19:21 +0400
 date		= 15/Jun/2002
 date_day	= 15
 date_mon	= Jun
 date_year	= 2002
 time		= 10:19:21
 time_hour	= 10
 time_min	= 19
 time_sec	= 21
 method		= GET
 request	= /~alex/cgi-bin/view.cgi?id=12
 req_user	= alex
 req_path	= /cgi-bin
 req_name	= view.cgi
 req_ext	= cgi
 req_qstring= id=12
 proto		= HTTP/1.0
 lstatus	= 200
 byte		= 3087

любые именя из первой колонки можно использовать для задания параметра `-gb`.
